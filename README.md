# JPMC_Automation_Demo
Demo for JPMC Client

Appium Setup on Windows: [Click Here](https://www.guru99.com/introduction-to-appium.html "Click Here")

Appium Setup on Mac: [Click Here](https://www.swtestacademy.com/how-to-install-appium-on-mac/ "Click Here")


### Execution Steps:

1.  Update the Capabilities in the src/test/configuration/Config.json file
2.  Set the Platform name in src/test/configuration/RunConfig.json file. e.g "webPortal_Dev", "Android" etc.
3.  Update the tags	name in src/test/java/stepDefinitions/RunCukesTest.java and Run the test.
